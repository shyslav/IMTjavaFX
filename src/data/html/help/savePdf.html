Задача управління запасами

Збереження відповіді в *.pdf формат

Що буде записано в файл?

В файл буде записана таблиця розв'язку, вхідні дані, та відповідь: мінімальні затрати та об'єми поставок. Таблця буде візуалізована, та мати приємний дизайн.

1) Потрібно ввети дані з програми або з текстового файлу

Для того, щоб переглянути як це зробити перейдіть у вкладку "Початок роботи"

2) У відкритій формі потрібно вибрати шлях куди зберігати файл

Виберіть в проводнику місце для збереження

3) Написати ім'я файлу

Вигадайте ім'я для вашого файлу

4) Натиснути кнопку "Зберегти"

При виконанні всіх дій, файл буде збережений.
